Music Generation

I co-developed an AI to compose music implementing a Markov Model to generate tunes. I then engineered an advanced Hidden Markov Model using Python libraries (music21, NumPy) trained on intricate music. This resulted in a successful system that generated coherent music with chords and rests, demonstrating the practical application of probabilistic models in creative AI.
